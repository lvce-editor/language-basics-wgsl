# Language Basics Wgsl

Wgsl syntax highlighting for Lvce Editor.

## Contributing

```sh
git clone git@github.com:lvce-editor/language-basics-wgsl.git &&
cd language-basics-wgsl &&
npm ci &&
npm test
```
